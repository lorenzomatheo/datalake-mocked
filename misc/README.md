# Misc

Notebooks esporádicos que não pertencem a um fluxo específico do pipeline de dados. São scripts utilitários, ferramentas de debug, operações ad-hoc e provas de conceito.

## Estrutura

```
misc/
├── clients/        - Notebooks ad-hoc para consultas em BigQuery e PostgreSQL.
├── dev/            - Utilitários para configuração do ambiente de desenvolvimento.
├── poc/            - Provas de conceito (ex: MedGemma, Vertex AI).
├── rfm/            - Análise e visualização de segmentação RFM de clientes.
└── *.py            - Scripts avulsos (operações, notificações, infra).
```
