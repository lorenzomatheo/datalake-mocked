# Squad Ativações

Este diretório contém os códigos relacionados ao squad de ativações da Maggu, responsável por integrar a maggu com plataformas externas para dar suporte ao time de ativação (que faz parte de Growth).

Algumas das integrações que já passaram por esse squad:

- **Airtable**: Foi utilizado como um "banco de dados paralelo" para armazenar informações de clientes e campanhas
- **Wassenger**: Utilizado para envio de mensagens em grupos do WhatsApp. É uma plataforma não oficial portanto foi descontinuado no segundo semestre de 2025
- **Hyperflow**: Provedor oficial de envio de mensagens via WhatsApp. Utilizado desde Julho de 2025
- **CustomerX**: Software de CSM utilizado para gerenciar o relacionamento com clientes desde Dezembro de 2025

Obs.: A pasta `postgres` contém automações importantes que alteram diretamente o banco de dados. Essa é a pasta mais delicada.
